Ancient Architecture Research Roadshow

古建筑调研：
木构、尺度与时间

这是一份适合直接路演的古建筑调研网页。它用网页而非传统 PPT 的方式，把对象价值、结构证据、调研方法与保护建议放在同一条叙事链里：一边保留纸本与展陈式阅读的厚重感，一边利用浏览器的滚动、动画、导航与可扩展性，把“研究过程”本身展示出来。

对象不是“景点”古建筑调研首先面对的是时代信息、构造逻辑与保存状态，而不是单纯的视觉欣赏。
网页不是“花哨外壳”它让章节跳转、热点注释、图像分层与证据链展示变得自然，尤其适合研究型路演。
01 · 调研总览

为什么古建筑调研特别适合用 HTML 做路演

因为这类内容天然包含“对象图像 + 构造判断 + 时间线 + 保护建议 + 图源说明”。PPT 很容易把它们拆散，而网页能把这些层次保留在同一阅读空间里，让观众更像在读一份展陈说明或研究版面。

01

网页表达的三点优势

  • 同屏容纳图像、注释和结论，阅读顺序更接近真实研究现场。
  • 滚动与导航形成空间化叙事，比“翻页”更适合讲时间与构造演化。
  • 后续可继续补图、补文献、补地图，直接演变成项目主页或数字展陈。

02 · 代表样本

从唐代木构到皇家空间秩序：四个典型对象

为了让观众快速建立认知，本次调研选择“早期木构个案 + 高层木塔 + 皇家城市空间”三种不同尺度的对象。它们共同构成了理解中国古建筑的重要入口。

南禅寺大殿常被视为中国现存最早的木构建筑实例之一。对它的关注，不只是因为“古老”，更因为它为理解早期木构尺度、构架与屋顶关系提供了稀缺样本。
Case A

南禅寺大殿：简洁尺度中的早期木构信息

路演时讲南禅寺，不应只说“唐代很早”。更重要的是让观众意识到：在一个体量并不巨大的木构中，我们能看到柱网、屋顶、斗栱与整体比例如何共同工作。它像一本被保存下来的“木构语法入门”。

这一页最适合讲“为什么少量样本也很重要”：因为当保存下来的早期木构极少，每一个对象都不是普通建筑，而是理解时代构造逻辑的关键证据。
Case B

佛光寺东大殿：大体量木构中的成熟秩序

佛光寺东大殿常用于展示唐代成熟木构的壮阔尺度。它和南禅寺对照起来看，意义就会非常清楚：前者像“早期语法样本”，后者则像“成熟写作范例”。从屋顶出檐、空间进深到整体体量，观众都更容易感受到唐代木构的宏大与稳定。

大体量 成熟构架 对照性极强
佛光寺东大殿的展示价值在于：它让“唐代木构”不再抽象，而变成可以被体量、进深、檐口与整体稳定感直接感知的空间经验。
应县木塔最适合在路演中承担“震撼点”。但震撼之后更重要的是说明：如此高的纯木结构，为何能长期存在？它依赖的不是某一个奇迹构件，而是整体结构体系与长期维护共同作用。
Case C

应县木塔：高层纯木结构如何成为结构史问题

应县木塔之所以在所有古建中格外有传播力，是因为它把“木”与“高”这两个词放在了一起。对研究型路演来说，最重要的不是只让人感到壮观，而是把“层级体系、稳定逻辑、修缮压力、长期保存”讲成一个结构问题。

当观众已经被高度和年代吸引时，下一步就要立刻把注意力从“奇观”转向“结构与保护”。这是研究型展示和旅游介绍最大的差别。
Case D

故宫：从单体建筑转向城市级空间秩序

当我们从木构个案转向故宫，研究尺度就发生了变化。此时重点不再是单体结构细部，而是中轴线、前朝后寝、等级秩序与礼制逻辑。故宫在路演中的价值，在于它提醒观众：古建筑研究从来不只研究构件，也研究空间如何组织权力与秩序。

故宫尤其适合网页展示，因为网页滚动天然有“轴线展开”的感觉。用纵向叙事讲中轴秩序，往往比横向翻页更有空间感。
核心洞见

古建筑研究最值得被看见的，不只是“它很老、很美、很壮观”，而是：它如何以结构、尺度、材料与空间秩序，持续保存了一个时代的建造逻辑。

03 · 调研方法

从现场到结论：一套适合答辩与路演的方法框架

研究型网页最能体现优势的地方，在于它能把方法本身讲清楚：你不是在“展示一堆图”，而是在展示你如何从图像、文献和现场观察走向判断。

03

对象确认

先明确研究对象的年代、类型、所在区域与保存背景，让问题定义足够清楚。

图像与测绘

通过立面、构造节点、轴测或平面信息建立对象的可视化证据基础。

文献互证

将现场观察与公开文献、博物馆说明或 UNESCO / 官方资料进行交叉核对。

保护判断

最后从结构、环境和展示传播三方面提出优先级明确的建议，而不是泛泛而谈。

因为古建内容本身就更接近“展板 + 图录 + 研究说明”的混合体，而网页可以天然容纳这种多层阅读结构。
